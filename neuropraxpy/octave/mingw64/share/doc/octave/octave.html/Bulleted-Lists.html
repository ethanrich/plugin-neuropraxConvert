11.11.2.6 Bulleted Lists

To create a bulleted list, type 

##
#
# * Bulleted list item 1
# * Bulleted list item 2
#

to get output like

  • Bulleted list item 1
  • Bulleted list item 2

Notice the blank lines, except for the ‘#’ or ‘%’ before and after the bulleted list!
